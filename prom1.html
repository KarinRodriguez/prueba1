Promedio 1

Proceso de Bitácora Digital

Descripción

A lo largo de las sesiónes podremos ver el proceso de nuestra página Web (Bitácora digital)
Aprenderemos a usar el programa visual studio code que nos ayudara como base en el desarrollo del proyecto.

Sesión 1

En la primera sesión descargamos el programa Visual Studio Code que es un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web.
Luego de descargar el programa cree el html con el nombre (Index.html) Iniciamos ccon poner la etiqueta de Inicio, Atributos, Valores, Contenido y Etiquetas.

Lista de cosas aprendidas en la sesión:

  • DOCTYPE = Indica al navegador cómo interpretar y renderizar el código HTML de una página web.
  • INDEX.HTML = Es el archivo principal de una página web, donde se encuentra todo el contenido que será mostrado en el navegador.
  • HEAD = Sirve como un contenedor para todos los elementos que se desean incluir en el encabezado de la página HTML.

Sesión 2

En la segunda sesión el docente nos enseño más formas de colocar nuestra información y así darle más cuerpo a nuestra página
y seguir con el desarrollo de nuestro Proyecto del Promedio 1.

Lista de cosas aprendidas en la sesión:

  • Title=Titulo
  • Body = todo lo que se va a visualizar dentro de la pagina texto, imágenes,etc.
  • ul=listas ordenadas y desordenadas.
  • ol= listas ordenadas
  • li= componentes de la lista
  • a href="xxxxxx" />a= colocamos la dirección web.

            Sesión 7

            En la séptima sesión aprendimos a hacer interacción con el puntero(mouse)
            Gestionamos nuestra pagina de manera mas ordenada teniendo encuenta las posiciones, cajas y celdas Tuvimos una introducción al Hosting, dominio (asociado a una dirección IP fisica) y componentes.

            Lista de cosas aprendidas en la sesión:

            • Height/width/border/margin/padding = Cambiar dimensiones a las Cajas(box).
            • :link = Permite dar estilo a un contenido no visitado.
            • :visited = Permite cambiar el estilo si un link ha sido clickeado.
            • :hover = Cambia la apariencia del botón cuando se pone el curso encima del elemento.
            • :active = Se aplica cuando se hace click en algún botón.
            • Empty cells, border spacing, border collapse = Tipos de celda para dar oden a nuestrs página.

              Sesión 8

              En la octava sesión conocimos 2 programas nuevos (Netlify) y (GitHub).
              Estos programas no ayudan a poder crear un hosting y un dominio para poder compartir el URL de nuestra página web.

              Lista de cosas aprendidas en la sesión:

              • Netlify = Plataforma de desarrollo que incluye servicios backend de creación, implementación y sin servidor para aplicaciones web y sitios web dinámicos.
              • GitHub = Se utiliza principalmente para la creación de código fuente de programas de ordenador.

                Sesión 9

                En la novena sesión tuvimos una introducción al programa Tinkercad de Autodesk.
                Introducción al Software : ARDUINO IDE - Diferencia de Electronica y Robótica,

                Lista de cosas aprendidas en la sesión:

                • Tinkercad = Tinkercad es un programa de modelado 3D.
                • Robótica = Tecnología que diseña y construye máquinas(robots).
                • Robot = Máquina programable capaz de realizar tareas de manera autónoma y semiautónoma.
                • Electrónica = Rama de la ciencia que estudia los fenómenos eléctricos y electrónicos.
                • Unidades de electrónica = Resistencia(R) Potencial(V) Corriente(I) Inductancia (L,M)

                  Sesión 10

                  En la décima sesión trabajamos con ARDUINO.
                  Realizamos un diseńo de circuitos programando y conectando cada componente a los (Digital y GND del Arduino)

                  Lista de cosas aprendidas en la sesión:

                  • DigitalWrite = Sirve para escribir un valor,estado o pin digital.
                  • Delay = Se usa para la ejecución del programa durante un período de tiempo específico, en milisegundos.
                  • Void Setup = Se utiliza principalmente para la creación de código fuente de programas de ordenador.
                  • DigitalRead(High-Low) = Lee los valores que se le da a un digital pin.
                  • PinMode = Sirve para configurar el pin como entrada (INPUT) o como salida (OUTPUT).
